GitHub Student Developer Pack

Не так давно парни с GitHub запустили очень интересный проект - GitHub Student Developer Pack, который позволяет студентам бесплатно использовать инструментальные средства профессиональных разработчиков.

Для того, чтобы получить данный "пакет" необходимо авторизоваться на сайте GitHub и на странице проекта заполнить форму(P.S. все поля нужно заполнять на английском). При заполнении потредуется прикрепить фотографию, которая доказывает то, что Вы являетесь студентом(например, фотографию студенческого билета).

PROFIT

  • Вам предоставляется возможность на GitHub иметь 5 приватных репозиториев до окончания обучения.
  • Вам получаете купон на 25$ на HackHands
  • Вам предоставляется возможность бесплатно использовать игровой движок Unreal Engine для консолей, PC и т.д. до окончания обучения.
  • Более подробно о предоставляемых бонусах можно узнать на сайте проекта GitHub Student Developer Pack

Немного о виртуальных машинах

В жизни порой возникают такие моменты, когда ваша ОС не удовлетворяет вашим требованиям (например, не хватает удобных приложений для работы с документами, нет подходящих редакторов, нет средств работы с языком программирования, который вы бы хотели изучить, неудобный графический интерфейс и т.п.). Неужели для этого нужно сносить свою текущую ОС и ставить другую (или делать dual boot)? На самом деле, перед тем, как делать что-то из этого, стоит дважды подумать, ведь другая ОС может не только дать вам что-то новое, но и что-то у вас отнять.

Для решения данной проблемы существуют виртуальные машины, которые позволяют "попробовать" другую ОС, либо полноценно работать с новой ОС, не удаляя текущей, к которой вы так привыкли. Вот некоторые из них: Virtual PC, VMware, VirtualBox, Parallels Desktop.

Так же существует программа, позволяющая использовать другую ОС без графического интерфейса (в основном какую-нибудь Linux-based ОС) - Vagrant. Vagrant работает поверх виртуальной машины (например, VirtualBox) и устанавливает свои, специально подготовленные образы из облака, которые вы можете скачать и развернуть при помощи того же Vargant (по сути, виртуальная машина просто является хранилищем для Vagrant, поэтому её включать не обязательно).

Тем, кто хочет поиграть с какой-нибудь Linux-based ОС, предлагаются следующие задания:

  • Установить VirtualBox
  • Установить на VirtualBox Xubuntu(либо какой-нибудь другой дистрибутив по желанию)
  • Научиться работать с эмулятором терминала(основные команды, которые можно изучить: man, cd, ls, cat, touch)

Инструментарий программиста ремесленника

У столяра есть свой рубанок, у юриста закон, у менеджера богатый дядя.

А чем пользуется обычный программист в своей повседневной работе? Этому и будет посвящена первая в этом сезоне лекция.
